Program — Accepted Papers

5 papers (3 long + 2 short) · 2 plenary speakers · half-day workshop

Session I

  1. [Long] Algorithm for Low-Variance Biclusters to Identify Coregulation Modules in Sequencing Datasets — Zhen Hu, Raj Bhatnagar
  2. [Short] Analysis of Obligate and Non-obligate Complexes using Desolvation Energies in Domain-domain Interactions — Mina Maleki, Md. Mominul Aziz, Luis Rueda

Session II

  1. [Long] Using Physicochemical Properties of Amino Acids to induce Graphical Models of Residue Couplings — K.S.M. Tozammel Hossain, Chris Bailey-Kellogg, Alan Friedman, Michael Bradley, Nathan Baker, Naren Ramakrishnan
  2. [Long] Analyze Influenza Virus Sequences Using Binary Encoding Approach — Hamching Lam, Daniel Boley
  3. [Short] A Lung Cancer Outcome Calculator Using Ensemble Data Mining on SEER Data — Ankit Agrawal, Sanchit Misra, Ramanathan Narayanan, Lalith Polepeddi, Alok Choudhary
